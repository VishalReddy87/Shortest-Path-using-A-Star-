# A* Path Finding Algorithm Visualization (Python + Pygame)

This project is a visual implementation of the A* (A-star) pathfinding algorithm using Python and Pygame.
It allows users to interactively create a grid, place start/end points, add obstacles, and watch how the A* algorithm finds the shortest path.
